一、判断正误

  1. ×

    有信号控制的城市道路交通流属于间断流，连续流的流量、密度、速度基本模型不适用。

  2. ×

    道路断面只能观测到经过这一断面的车辆数据，包括流量、车头时距、车头空距等信息。而车流密度必须要观测一整段路面中的车辆数才能得出结论。

  3. √

    根据速度-密度关系

    当流量接近于0，则车流密度也接近于0，即K = 0时，代入式子可知。即速度接近自由车速

  4. √

    服务水平的影响因素包括交通条件，交通条件中包括车辆类型和方向分布和车道分布。所以服务水平影响因素中包括车辆类型。当系统中存在重车时，会对系统造成和当量小汽车造成不同的影响，所以需要重车修正。

  5. √

    服务水平可以由密度查表得知。

  6. √

    时间序列预测法根据过去的数据进行未来的预测，可以反映事物的因果关系

二、简答题

1、试解释信号控制交叉口的总损失时间的构成。

总损失时间包括启动损失和清空损失。

启动损失：通常位于队首的前4辆车由于启动反应时间 、加速时间，其车头时距会大于饱和车头时距，这一 多余的时间称为启动损失。

清空损失：当一个流向停止通行时，为保证通行安全，在放行冲突流向之前需要一段清空时间， 这一时间间隔称为清空损失时间

 

2、简述交通量的时间、空间变化特征及其应用。

交通量的时变特性：

  • 高峰与平峰
  • 时变性：小时内的变化，分钟交通量、15分钟交通量以及高峰小时系数等
  • 月变性：一年中各月交通量的变化
  • 周变性：一周七天交通量的变化
  • 日变性：一日内交通量的变化
  • 交通流时变性之间的关系：年平均日交通量、月变系数、周变系数、日变系数等

通过这些变量可以确定在一段时间内的变化幅度以及高峰小时系数PHF等，从而促进交通规划、设计、建设与管理的科学性。

交通量的空间变化特性：

  • 城市不同区域交通流分布
  • 不同功能道路上交通流分布
  • 路段交通流分布、交叉口交通流分布
  • 不同车道上分布：里车道、中间车道、外车道
  • 方向性交通流分布：潮汐分布

应用：利用交通流的空间变化特性，可以更好的规划道路设计。对于城市内不同区域内的不同功能道路，都要根据交通量的不同进行不同的容量设计。

 

3、简述排队系统的排队规则。

排队规则指到达的顾客根据怎样的规则接收服务

包括：

  • 损失制：顾客到达时若没有空闲的服务台则离开
  • 等待制：顾客到达时若所有服务台被占据，则进行排队等待
  • 混合制：顾客到达时若队伍长于L，则离开，否则进行排队等待

 

4、匝道合流区通行能力的计算中需要考虑的主要因素和步骤。

要考虑的因素：

  • 上游车道的总流量
  • 匝道的需求流量
  • 加速车道的长度
  • 匝道的自由流速度
  • 距离上游最近匝道的距离
  • 距离下游最近匝道的距离
  • 合流区上游车道数量
  • 高峰小时系数PHF
  • 重车修正系数
  • 驾驶员修正系数

  4. 最后合流区的总通行能力就等于上游1、2车道流率与进口匝道需求流率之和

5、简述供需平衡分析框架及各组成部分的内容。

基本供需平衡分析框架

需求输入：

  • 微观层面：单个车辆到达的时间
  • 宏观层面：一定时间间隔内车流的到达率
  • 不饱和条件下，需求小于供给能力，可以分析多个时间段
  • 饱和条件下，需求超过供给能力，应该分析多个时间段

供给输入：

  • 微观层面：能够服务的最小车头时距
  • 宏观层面：一定时间间隔内的最大车流率

控制输入：

  • 不受时间影响，如：跟车规则、服务规则（先进先 出）、车辆的性能等等
  • 基于时间的控制，如：交通信号、高峰时期的交通 管制
  • 与交通相关的控制，如铁路与道路交叉口、匝道控 制、交通信号、驾驶员在不良情绪下调整驾驶行为

环境输入：

  • 包括：能见度、天气状况、路面状况、导致注意力 分散的因素等

分析步骤：利用上述输入数据估计交通系统的性能

  • 微观层面包括：跟车理论、时空图分析法、随 机排队论、微观仿真
  • 宏观层面包括：通行能力分析、流密速关系、 交通波分析、确定排队论、宏观仿真

性能：

  • 交通系统使用者：行程时间多少、延误大小、排队长 短、舒适度、能源消耗等方面
  • 交通系统管理者：供给能力、服务水平、车辆排放、 噪声大小、安全度、综合交通成本

反馈过程：

  • 确定供需平衡
  • 解决“如果”的情况
  • 敏感性分析
  • 最优化目标函数
  • 调查使用者对系统改善的反应

三、论述题

1、目前自动驾驶汽车的技术研究非常热，如果未来城市道路上所有车辆都实现了自动驾驶，那么在分析道路通行能力、交通流三变量（流量、速度、密度）的关系、车队离散等方面的分析与现在的分析会有什么不同？

答：

道路通行能力：

  1. 首先由于所有的车辆都是自动驾驶，所以在计算15分钟当量小客车流率时就不存在驾驶员修正系数，而是另一个对于自动驾驶系统进行评判的系数

  2. 人的反应时间是很难改变的，但是自动驾驶系统随着技术的发展，反应时间可能会超过人类。所以在高峰时刻，车流的车头空距可能更小，因为自动驾驶车辆可以在更短的时间内进行紧急情况的处理。

    车头空距变小，道路的通行能力就会更大。

  3. 然后不要考虑交通环境对人的影响。比如雨雪天导致实现不清楚，驾驶员的视觉特性对停车视距的影响等等。取而代之的是各种交通环境对自动驾驶车辆感知系统的影响，比如雪天的白色对于摄像头的影响等。

交通流三变量：

  1. 在自动驾驶环境下，常规的格林希尔治模型不一定准确。需要再进行新的调查确定新的模型
  2. 速度的变化。自动驾驶环境下车流速度的变化都是由驾驶系统确定的，而不是之前由人随机确定。从而可以制作更精确的速度变化模型。

车队离散：车队离散是非常好判断的。

  1. 一种情况是交通基础设施如红绿灯可以与车流进行通信从而确定车流速度，再而确定下一个路口的绿灯到来时间，形成一个连续的绿灯流。
  2. 另一种情况是基础设施已经知晓各种自动驾驶车辆的处理方式，从而预测接下来的车流情况。

 

2、系统评价是系统工程中的一个重要环节，在系统的规划、设计、管理过程中，对备选方案进行优劣排序，为最优方案的选择决策提供依据。系统评价应用于交通系统即为交通系统评价。请列举系统评价的主要方法，并论述交通系统评价可能的难点及相应的对策？

答：

系统评价的主要方法有：

  • 费用-效益分析法
  • 关联矩阵法
  • 层次分析法
  • 模糊综合法
  • 聚类分析法

难点和对策：

  1. 多种评价方法都要用到系统评价各个指标所占的权重从而建立评价矩阵，权重的确定是难点之一。

    对策是要经过多方面调查，既要考虑专家的意见，又要调查当地民众的意见，还要参考往年相关工程的经验进行抉择。

  2. 方法的选取

    系统评价有着多种方法。对于同一个系统，可能不同的评价方法评价出来的最优方案是不一样的。

    这时候就需要根据不同方案的优缺点并结合系统的特点进行方案的选择。或者对不同的评价方法再进行权重分析，然后再将每个方案乘上不同的权重，最后确定最优方案。

  3. 模糊集合的量化问题。大多数系统指标难以用确定的数字衡量，只能给出一个模糊的评价语，需要对不同评价语进行量化。量化的标准也是难点之一。

    这时需要结合大量的往年数据进行考证。

  4. 评价方法的反馈问题

    当有多个方案时，根据已有的评价方法确定一个最优的方案。但是无法根据已有方案的实施确定这是否实际是最优的方案。所以系统评价的反馈也是难点之一。

    这时需要对于小型工程可以按照所有方案进行一遍对比实验。然后对评价方法进行修改。
